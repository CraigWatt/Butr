How it works

No in-app chat needed

  1. Connect your Trading 212 API key to Butr securely.
  2. Use any AI chat (ChatGPT, Claude, etc.) to describe your trade intent.
  3. Butr creates a portfolio-aware preview with validation checks.
  4. Approve the ticket. Only then does anything execute.

Why Butr

Control without complexity

  • Human-in-the-loop: every trade requires your approval
  • Portfolio-aware: checks positions before suggesting
  • Bring your own AI: works with any chat client
  • UK-focused: built for ISA compliance

What Butr is not

Staying focused

  • No autonomous trading or robo-advisor
  • No social trading or copy features
  • No portfolio dashboard (use Trading 212 for that)
  • No live execution in v1 (approval tickets only)
