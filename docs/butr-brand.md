# Butr Brand Notes

Butr is a calm investing copilot for UK ISA users.

## Brand shorthand

- smooth investing, sharper decisions
- calm capital copilot
- approval-first investing control

## Must-haves

- butter-yellow logo cue
- friendly but restrained UI
- serious backend, light surface
- ISA-only for V1

## Must-avoids

- quant bro styling
- crypto energy
- dark terminal-first UI
- aggressive red/green trading app vibes
- cluttered analytics dashboards
